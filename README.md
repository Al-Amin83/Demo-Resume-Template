# 📄 Personal Resume — Md. Al-Amin Hossain

A personal demo resume built with pure HTML, designed to showcase my education, technical skills, and projects as a Full Stack Web Developer. Features a clean two-column layout with a dark sidebar, timeline-based sections, and a responsive design — no frameworks required.


## 🚀 Features

- Responsive two-column layout (sidebar + main content)
- Timeline-style Work Experience and Education sections
- Skill tags, language proficiency bars, and core competency cards
- Google Fonts: DM Serif Display + Inter
- Pure HTML — no frameworks, no JavaScript
- Mobile-friendly design

---

## 🛠️ Built With

- Google Fonts

## ✏️ Customization

To make this resume your own:

| What to change | Where to find it |
|---|---|
| Name, bio, contact info | `Resume.html` — Hero and Sidebar sections |
| Profile picture | Replace `Profile.jpg` with your own photo |
| Skills & experience | `Resume.html` — Timeline and Tag sections |
